HTML Intro

Block elements default to having a new line before and after the element. By contrast, inline elements do not start with a new line. In standard practice, an inline element must be nested in a block element, as it cannot stand alone on a page.
The span element is an inline equivalent of a div element. It is most commonly used to apply CSS styling to all of the text it contains, without applying those styles to the other text in the block.
